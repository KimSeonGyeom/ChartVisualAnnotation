# Chart Images

Place your chart images in this folder. The images should be referenced in `src/config/stimuli.json`.

## Expected Images

Based on `stimuli.json`, you need the following images:

1. `chart_example_1.png` - Sales chart
2. `chart_example_2.png` - Temperature chart  
3. `chart_example_3.png` - Comparison chart
4. `attention_check_chart.png` - Chart with clear title for attention check

## Image Guidelines

- **Format**: PNG or JPG recommended
- **Size**: Recommended 800x600 pixels or similar aspect ratio
- **Content**: Clear charts with visible titles, labels, and data
- **Attention check**: The attention check chart should have a clearly visible title at the top

## Example Chart Sources

You can create charts using:
- Excel/Google Sheets
- Python (matplotlib, seaborn)
- R (ggplot2)
- Online chart makers

Make sure your charts match the captions defined in `stimuli.json`.
